## 🌍 Deployment (Netlify)

### Manual Deployment

1. Log in to Netlify
2. Select **Add new site → Deploy manually**
3. Drag and drop the project folder
4. The site will be published instantly

### Git-Based Deployment

1. Push the project to GitHub
2. In Netlify, choose **Add new site → Import from Git**
3. Connect your repository
4. Deploy

Netlify will automatically update the site on every push.
